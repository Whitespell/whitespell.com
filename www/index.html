How it works

Whitespell has a whole bunch of awesome features but the most important is probably the way it works in its most basic form. Lets create a little script that allows us to insert todos into the database.

#1 Creating a Prepaired Statement

The part where Whitespell really starts to shine is with Prepared Statements. Think of prepared statements as dynamic SQL queries. They are simple to write but very powerfull and secure. They think intelligently about your data and the requests you perform to them and optimise your database accordingly. Lets create one using our CLI.

ws todo_app ps:new
name: insert_todo
public?: y
query: INSERT INTO todos (content) VALUES ('$this->{content}')

✓ Prepared Statement 'insert_todo' created

var ws = new Whitespell('YOUR_API_KEY');

ws.send({
  'task': 'insert_todo',
  'content': 'Do Groceries'
});

ws.on('insert_todo', function(data){
  console.log(data.name+' was inserted!');
});
